## Fonts

The used font on the site is `Nunito` from Google Fonts API


In HTML
```html
<link href="https://fonts.googleapis.com/css?family=Nunito&display=swap" rel="stylesheet">
```

In css
```css 
body {
    font-family: 'Nunito', sans-serif;
}
```
